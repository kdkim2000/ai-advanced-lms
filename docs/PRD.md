## 현재 기능 (As-Is)

### 학습 기능
- **MDX 챕터 리더**: 서버 사이드 렌더링, Shiki 코드 하이라이팅, 커스텀 Callout 컴포넌트
- **챕터 완료 체크**: 즉시 완료/미완료 토글, 모듈 진도율 자동 계산
- **북마크**: 챕터 즐겨찾기 토글, 대시보드에서 확인
- **노트**: 챕터별 텍스트 노트 저장 (localStorage)
- **TOC**: MDX 헤딩 자동 추출 목차

### 퀴즈 기능
- **모듈별 퀴즈**: 모듈당 30~324문제, 문제 네비게이션 그리드
- **세션 저장/재개**: 중단된 퀴즈를 다음 방문 시 이어서 풀기
- **실전 문제풀이**: 전체 모듈에서 랜덤 20문제 선별
- **오답노트**: 틀린 문제 자동 저장, 재풀기 시 정답 시 solved 처리
- **결과 화면**: 점수, 등급(A-F), 모듈별 정확도

### 대시보드
- 전체 진도율 + 완료 챕터/모듈 수
- 모듈별 진도 링크
- 최근 완료 챕터 이력 (최대 5개)

### 설정
- 라이트/다크/시스템 테마 전환

### UI/UX
- Ctrl+K 검색 팔레트
- 사이드바 접이식 모듈 네비게이션
- 반응형 (모바일/태블릿/데스크탑)
- Apple 미니멀 디자인 (투명 카드, CSS 변수 기반 테마)

## 리뉴얼 목표 (To-Be)

### 디자인 시스템 — OPUS-X (완료)

Claude design 핸드오프(`docs/design_handoff_ai_advanced_lms/`)를 기반으로 Samsung SDS OPUS-X 디자인 시스템 적용 완료.

- 보라 브랜드 컬러 (`#6941C6`) 전면 적용
- Noto Sans KR 로컬 폰트 + Sora 헤딩 폰트 도입
- 솔리드 카드 배경 + `shadow-xs` 그림자 계층 도입
- `[data-theme="dark"]` 선택자 기반 완전한 다크 모드
- 모듈별 6가지 카테고리 색상 시스템
- 176개 CSS 변수로 모든 색상 토큰화
- 자세한 내용: `docs/DESIGN.md`

### 학습 UX 개선
- **인터랙티브 Python 실행**: 챕터 내 Pyodide 코드 블록 통합 (현재 코드 존재하나 미연결)
- **학습 진도 시각화**: 모듈 진도 히트맵 또는 스트릭 표시
- **챕터 예상 소요 시간**: frontmatter에 `duration` 필드 추가

### 퀴즈 UX 개선
- **난이도 뱃지**: `level` 필드(LV.1~LV.3) 시각적 표시
- **카테고리 필터**: `category` 필드 기반 문제 필터링
- **점수 기반 배지**: 70% 이상 시 합격, 90% 이상 시 우수 배지

### 모바일 최적화
- 퀴즈 문제 네비게이션 그리드 모바일 스크롤 개선
- 사이드바 드로어 모드 강화

### 로드맵 (미래 기능)
- 수료증 발급 (모듈 100% + 퀴즈 70% 이상 시)
- 학습 통계 API 연동 (현재 localStorage 한계 극복)
- 스페이스드 리피티션 오답노트 알고리즘
